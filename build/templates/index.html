    Welcome to Circa

    Circa is a code name for this web app. It's easier than saying "the centralized ip-based reboot center application". Circa makes rebooting multiple devices easy and saves time. Simply select the building or individual devices that need to be reset, click the reboot button and let Circa do the rest for you.

    How It Works

    Circa is built using Flask which is microframework for Python that allows for the rapid development of web applications. Flask serves up a form that is hard coded with the ip information of your devices. When selected the form sends a POST request to the url that runs a factory cgi script that safely reboots the device. Once this POST request returns if the function exited successfully or not it will return the form with reports at the top. One report will be in green, being the list of successfully rebooted devices. The other will be in red as they have failed for one reason or another.
